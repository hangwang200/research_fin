# 🎯 Purpose · LLM Wiki 的存在理由

> **本文档的角色**：定义 Wiki 的「**为什么**」与「**朝哪里去**」。
> [[Schema]] 回答 "How does the wiki work?"；[[Purpose]] 回答 **"Why does it exist?"**
> LLM 在每次 [[Ingest]] / [[Query]] 时**必须先读本文件**，确保所有产出都服务于核心目标，而非「为信息而信息」。

---

## 1. 北极星问题（[[North Star Question]]）

> **「在 [[AI]] 重塑世界的十年里（2025-2035），如何用第一性原理识别真正具备 [[长期复利]] 能力的优质资产，并以合理价格长期持有？」**

这个问题决定了：
- ✅ 我们关心 **[[Compounders]]**（持续复利的好生意），不关心 [[Cigar Butt]]（捡烟蒂套利）
- ✅ 我们关心 **5-10 年视角**，不关心季度噪音
- ✅ 我们关心 **[[Free Cash Flow]] + [[ROIC]]**，不关心 [[Stock Story]]
- ❌ 我们不做 [[Day Trading]] / [[Technical Analysis]] 短线
- ❌ 我们不投 [[Memecoin]] / [[纯主题炒作]]

## 2. 三大研究战场（[[Research Pillars]]）

### 🤖 战场一：[[AI Infrastructure]]（核心优先级）

**为什么**：[[Generative AI]] 是 [[Internet]] 之后最大的科技 [[范式转移]]。2025-2030 [[Hyperscaler]] [[CapEx]] 累计将达 **$3-4 万亿**，是有史以来最大规模的资本周期之一。

**关键问题**：
- 🥇 **谁在卖铲子？**（[[Nvidia]] / [[TSMC]] / [[ASML]] / [[Broadcom]] / [[SK Hynix]]）
- 🥈 **谁在挖金矿赚到真钱？**（[[Hyperscaler]] 中 [[META]] / [[Microsoft]] / [[Google]] / [[Amazon]]）
- 🥉 **谁会被淘汰？**（[[Legacy SaaS]] / [[Stack Overflow]] / [[传统搜索]] / [[长尾内容平台]]）

**重点跟踪指标**：
- [[CapEx Intensity]] = CapEx / Revenue（健康区间 25-35%，超过 40% 警惕）
- [[GPU Utilization]] / [[Token Economics]] / [[Inference Cost Curve]]
- [[Model Capability]] 进化曲线 vs [[Compute Scaling Laws]]

### 💰 战场二：[[Quality Investing]] / 优质消费 & 平台

**为什么**：高 [[ROIC]] + 强 [[Network Effects]] 的平台公司是 [[Buffett]] / [[Munger]] 模型最甜蜜的领域。

**关注子领域**：
- **[[OTA]] / 在线旅游**：[[TCOM]] / [[Booking]] / [[Expedia]] / [[Airbnb]]（[[Travel as Service]]）
- **[[E-commerce]]**：[[Amazon]] / [[Pinduoduo]] / [[BABA]] / [[Sea]] / [[Mercadolibre]]
- **[[Digital Advertising]]**：[[META]] / [[GOOGL]] / [[ByteDance]] / [[Snap]]
- **[[Premium Consumer]]**：[[AAPL]] / [[Costco]] / [[Hermès]] / [[贵州茅台]]

**重点跟踪指标**：
- [[ROIC]] - [[WACC]] 价差（> 10pp 视为价值创造）
- [[Take Rate]] / [[ARPU]] 演变
- [[Pricing Power]] 证据（敢提价不掉客户）

### 🌍 战场三：[[Macro]] & [[China Exposure]]

**为什么**：宏观决定 [[Beta]]，[[Geopolitics]] 决定 [[Tail Risk]]。中概股在 [[2022 低点]] 后的均值回归仍未结束，但 [[US-China Tech War]] 风险常在。

**关键问题**：
- [[Fed Rate Cycle]] 何时反转？对成长股估值的影响
- [[China Stimulus]] 是否能重新激活 [[消费复苏]]？
- [[Tariffs 2026]] / [[Trump 2.0]] 政策对供应链的重构
- [[VIE Structure]] 退市风险 vs [[港股双重上市]] 缓冲

## 3. 评估框架（[[Decision Framework]]）

每个公司分析的最终输出必须能回答 [[Charlie Munger]] 三连问：

| 问题 | 标准 | Wiki 中的证据来源 |
|---|---|---|
| **Q1: 这是不是一门好生意？** | [[Wide Moat]] + ROIC > 15% + [[Operating Margin]] 持续 | [[Templates/Company-Analysis-Playbook]] §1-2 |
| **Q2: 价格是否合理？** | [[Margin of Safety]] ≥ 30% vs [[Intrinsic Value]] | Playbook §5（DCF + 估值带）|
| **Q3: 管理层是否值得托付？** | 创始人控股 + 资本配置纪律 + 长期主义 | Playbook §1.5 + §4 |

**仓位决策规则**：
- 三问全 ✅ + 安全边际 > 30% → **Strong Buy**（核心仓 5-10%）
- 两问 ✅ + 安全边际 > 0% → **Buy / Hold**（卫星仓 1-3%）
- 一问 ❌ → **Watch**（不进仓）
- Q1 ❌（生意不好）→ **永远不投**，无论多便宜

## 4. 关键投资论点（[[Living Theses]]）

> 这些论点会随着研究演化。每次新增公司分析或宏观判断时，必须**回头检验**这些论点是否仍成立。

### Thesis 1: AI CapEx 周期 = Hyperscaler 的「炼狱 + 冠冕」

- **正面**：先发者的 [[AI Moat]] 会越拉越大（[[OpenAI]] / [[META]] / [[GOOGL]]）
- **风险**：CapEx 黑洞拖累 FCF（[[META]] FCF 已 YoY -16%）
- **观察**：等 2026-2027 看哪些 Hyperscaler 的 [[ROIC]] 真正企稳回升
- **状态**：🟡 待 [[GOOGL]] / [[MSFT]] / [[AMZN]] 完成对比验证

### Thesis 2: 中国消费 / 旅游业是被错杀的优质资产

- **正面**：[[TCOM]] / [[Pinduoduo]] / [[BABA]] 估值远低于历史均值，但 [[ROIC]] / [[OCF]] 仍健康
- **风险**：[[消费疲弱]] / [[反垄断]] / [[VIE]] 退市
- **观察**：跨境游恢复 + [[出境游]] 数据
- **状态**：🟡 待 [[TCOM]] 完成

### Thesis 3: 创始人控股的科技公司具有结构性 Alpha

- **逻辑**：[[Mark Zuckerberg]]（META 61% 投票权）、[[Jensen Huang]]（NVDA）、[[Elon Musk]]（TSLA）等创始人 CEO 能做反 [[共识]] 的长期决策
- **正面证据**：[[META]] [[Year of Efficiency]]、[[Reality Labs]] 长期下注
- **反面**：治理无制衡（[[Reality Labs]] $80B+ 累计亏损是 Zuck 个人意志的产物）
- **状态**：🟢 部分验证（META），需更多样本

### Thesis 4: 估值要看 [[Owner Earnings]] 而非 [[GAAP Net Income]]

- **逻辑**：[[GAAP]] 净利润经常被一次性税款、[[商誉减值]]、[[SBC]] 扭曲
- **示例**：[[META]] FY2025 GAAP 净利 -3% 但实际利润 +27%（剔除 $25.5B 一次性税款）
- **执行**：所有公司分析必须算出真实 [[Owner Earnings]] = OCF - 维持性 CapEx
- **状态**：🟢 已纳入 [[Templates/Company-Analysis-Playbook]] §3

## 5. 反目标（[[Anti-Goals]]）

> 明确**不做什么**比做什么更重要。

| ❌ 不做 | 原因 |
|---|---|
| 不追逐 [[热门题材]] / [[概念股]] | 没有现金流支撑的故事终将破灭（参考 [[NFT]] / [[Metaverse 一阶段]]） |
| 不做季度财报短线交易 | 时间是好生意的朋友，是平庸生意的敌人 |
| 不投自己看不懂的（[[Circle of Competence]]）| 半导体设备细节、生物医药机理 → 列入「Pass」 |
| 不为「便宜」而买 | [[Value Trap]] 比错过好机会更伤害复利 |
| 不依赖 [[Stock Tip]] / [[内幕]] | 唯一可持续的优势是研究深度，不是信息差 |
| 不在 Wiki 中堆砌信息 | 每页都必须服务于「决策」 —— 无法回答 Munger 三问的页面要么删掉，要么深化 |

## 6. 个人偏好（[[Investor Personality]]）

- **风险偏好**：中等偏低 —— 优先保本，再求收益（[[Buffett Rule #1]]）
- **持仓周期**：3-10 年（[[Long-term Investor]]）
- **行业偏好**：科技 + 消费 > 金融 > 能源 > 医疗（[[Circle of Competence]]）
- **地域偏好**：美股 50% / 港股+A 股 30% / 现金 20%（动态调整）
- **能接受的最大回撤**：单仓 -50%，整体组合 -25%
- **拒绝接受的**：永久性资本损失（公司基本面崩塌而非市场情绪）

## 9. 与 LLM 的协议（[[Human-LLM Contract]]）

**人类负责**：
- 选源（哪些公司值得分析、哪些研报值得读）
- 提问（什么是真正重要的问题）
- 判断（最终的投资决策、仓位管理）
- 演化本 Purpose（每季度复盘修订）

**LLM 负责**：
- 阅读所有源、抽取关键事实
- 维护 [[index.md]] / [[log.md]] / [[overview.md]]
- 在公司分析中严格执行 [[Templates/Company-Analysis-Playbook]] 方法论
- 主动指出 [[矛盾]] / [[Knowledge Gap]] / [[红链]]
- **绝不**编造数据；**绝不**省略来源；**绝不**给出无证据的结论

**协议红线**：
- LLM 写的任何数字必须有 [[来源]] 引用（一/二/三级源分级）
- LLM 不得在未读取一手财报的情况下生成 Section 3 / 5
- LLM 必须在不确定时**明确说出不确定**（避免幻觉）
