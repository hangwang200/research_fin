## 🎯 当前已确立的观点

> 这部分会随着研究演化。每次新分析 / 新论点登记到 [[purpose.md]] §4 后，同步更新此处摘要。

### 1. [[META]] 是 [[AI Infrastructure]] 战场的优质标的（信心：🟢 高）
- **逻辑**：广告业务 [[ROIC]] > 33%、[[Forward PE]] 17.7x、[[Mark Zuckerberg]] 长期主义经营哲学（[[Year of Efficiency]] 已证明）
- **风险**：FY2026 [[CapEx]] $115-135B 是有史以来最大豪赌，[[Reality Labs]] 累计亏损 $80B+
- **决策**：Buy on Dip / Hold（核心仓 5-10%）
- **触发重审**：单季度营收 YoY < +10% / [[FTC]] 拆分判决 / CapEx 超 $150B

### 2. [[TCOM]] 是 [[China Reopening]] + [[Inbound Travel]] 的最纯标的（信心：🟢 高）
- **逻辑**：[[Trip.com Alliance]] 70% [[OTA]] GMV 寡头垄断；FY2025 营收 +17%、国际业务 [[Bookings]] +60%；净现金 RMB 85.7B 占市值 38%；新 $5B [[Buyback]] = 市值 ~15%；Forward PE 12-14x 处近 5 年 < 25 分位
- **风险**：[[SAMR Anti-Monopoly Investigation]] 罚款上限 ≈ RMB 5.3B + 强制整改（[[最低价]] / [[二选一]] 取消）；[[Min Fan]] / [[Qi Ji]] 共同创始人 2026-02 退出董事会；S&M 费用 +25% > 营收 +17%
- **决策**：Buy on Dip / Hold（核心仓 3-5%）
- **触发重审**：[[Q1 2026 Earnings]] (5/19) 核心营业利润 YoY < +5% / [[SAMR]] 罚款 > RMB 5B / 国际 OTA 增速 < +30%

### 3. [[Owner Earnings]] > [[GAAP Net Income]]（信心：🟢 高 · 双案例验证）
- **证据 1**：[[META]] FY2025 GAAP NI -3% 但真实利润 +27%（剔除 $25.5B 一次性税款）
- **证据 2**：[[TCOM]] FY2025 GAAP NI +95% 是「**双倍幻觉**」—— 剔除 RMB 19.9B 一次性投资公允价值变动后，**核心经营净利润反而 -12%**（S&M +25% 跑赢营收）
- **共同模式**：**两个完全相反的会计 noise 方向**（META 用一次性税款掩盖好消息，TCOM 用一次性投资收益放大坏消息），都需要用 [[Owner Earnings]] 视角穿透
- **执行**：所有公司分析必须算真实 [[Owner Earnings]] = OCF - 维持性 CapEx
- **已固化于**：[[Templates/Company-Analysis-Playbook]] §3

### 4. [[Charlie Munger]] 三连问是终极决策框架（信心：🟢 高）
- **应用**：所有公司分析必须能回答 ① 是不是好生意 ② 价格是否合理 ③ 管理层是否值得托付
- **已固化于**：[[Templates/Company-Analysis-Playbook]] §4 + [[purpose.md]] §3
- **跨案例发现**：[[META]] 三连问全 ✅；[[TCOM]] 三连问 2.5/3 ✅（治理结构「分散持股 + 创始人持股低」是结构性扣分项）

### 5. [[Geo Discount]] vs [[Quality]] —— 估值需要分层（信心：🟡 中）
- **观察**：[[META]] Forward PE 17.7x，[[TCOM]] Forward PE 12-14x；同样是 [[Wide Moat]] / [[现金牛]]，差距 30-40% 来自「中国 [[Geo Discount]]」+ [[SAMR]] 监管溢价
- **隐含问题**：是否存在「便宜的好生意」陷阱？— [[BABA]] / [[BIDU]] 长期 PE < 15x 仍未跑赢
- **下一步验证**：补 [[GOOGL]] / [[BABA]] / [[Pinduoduo]] 形成完整对照

### 6. **[[Quality Investing]] 弹药库已成型**（信心：🟢 高 · 方法论级）
- **新建 7 大核心概念**（2026-04-26）覆盖 4 个分析维度：
    - **盈利能力**：[[ROIC]]（双口径） + [[Owner Earnings]]（Buffett 1986）
    - **护城河**：[[Wide Moat]]（Pat Dorsey 5 大来源 + 7 Powers）
    - **资本结构**：[[Tangible Net Worth]]（Graham 安全边际） + [[CapEx]](Maintenance vs Growth)
    - **运营质量**：[[流动比率]] + [[应收应付比]]（Pricing Power 间接证据）
- **方法论统一**：每个概念页都包含「公式 + 红线 + META/TCOM 实测案例 + 经典文献」四件套，可被未来 ingest 直接复用
- **跨案例洞察**：[[TCOM]] AR/AP 0.30x 揭示了「[[OTA Float]]」结构（RMB 50B+ 供应商免费融资）—— 这是表层 ROIC 11% 之下的**真实超额经济价值**，未来分析 [[BKNG]] / [[Expedia]] / [[Meituan]] 时直接对照
- **下一步**：[[Network Effects]] / [[Switching Cost]] / [[Pricing Power]] 是 [[Wide Moat]] 5 大来源中可独立成页的子概念，应在下一批补全
